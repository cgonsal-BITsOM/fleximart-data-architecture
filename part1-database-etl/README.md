## Part 1 overview
This repository implements an end‑to‑end data engineering solution for FlexiMart to transform raw, low‑quality CSV data into a clean, relational analytics‑ready database and answer core business questions. The solution covers ETL automation, normalized schema design, and business analytics SQL, supporting repeatable, auditable data operations.

1. ETL Pipeline (etl_pipeline.py):  Automated extraction, cleansing, transformation, and loading of raw CSV data into PostgreSQL.
2. Database Schema Documentation (schema_documentation.md) : Define and explain the logical and physical data model.
3. Business Queries (business_queries.sql): Deliver actionable insights from curated data.
